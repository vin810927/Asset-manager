# 個人資產管理 Dashboard

這是一個純前端的個人資產管理工具，目標是協助追蹤資產、負債、持股曝險與資產配置比例。

## 目前功能

- 新增資產與負債項目
- 自動計算總資產、總負債、淨資產
- 計算各資產類別比重
- 偵測單一標的是否超過設定上限
- 產生簡單再平衡提醒
- 使用瀏覽器 localStorage 儲存資料
- 不需要後端、不需要資料庫

## 使用方式

1. 開啟 `index.html`
2. 輸入資產或負債資料
3. 點選「新增」
4. Dashboard 會自動更新

## 資料安全

目前所有資料都儲存在使用者自己的瀏覽器 localStorage 中。

資料不會上傳到任何伺服器。

若清除瀏覽器資料，內容可能會遺失。

## 適合部署方式

可直接使用 GitHub Pages 部署。
